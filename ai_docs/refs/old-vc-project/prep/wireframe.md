### Navigation Flow Map

```
Landing → Signup → /dashboard ||
           ↓ Login → /dashboard
/dashboard → Investments → Investment Detail
           → Fund Reports → Upload Form → Processing Queue
           → Co-investors → Co-investor Profile → Deal Room
           → Analytics → Detailed Analytics View
           → Profile → Billing → Subscription |→ Payment History
/dashboard → Admin Panel → Users → User Detail → Edit User
                        → Analytics → System Metrics → Export Report
                        → Content → Fund Formats → Format Config
                                 → AI Models → Model Settings
                                 → Documentation → Article Editor
                                 → Announcements → Announcement Builder
```
